# 規則性か自然性か——ヴァール＝イェスペルセン討議（1935年）

## 要約

この討議は五つの節と共同結論から構成されている。

**I. 序論。** 両著者はまず共通の立場を確認する。いずれも自然主義派に属し、純粋なア・プリオリ的な企て（ロ語）、ヴォラピュクの歪み、エスペラントの人工的要素（代名詞一覧表、*edzo* のような造語、硬直した語尾体系、ハット付き文字）を斥け——ただしザメンホフ考案の前置詞 *ye* はともに称賛する。次いで各自の指針を述べる。ヴァールは、補助語は欧州言語に共通する国際的形式に基づいた有機的・自律的な全体であるべきで、予備知識なしにほぼ理解できるものでなければならないと主張する。イェスペルセンは学習の容易さを最優先し、最大限の規則性と単純さを求めながら、国際性と表現力も認める。

**II. アルファベット。** ヴァールは、ノヴィアルが以前 *c* と *z* を *k* と *s* に置き換えたことを語源的一貫性の破壊として批判する。生涯を音声学に捧げたイェスペルセンはその音声論的根拠を擁護しつつも、一般公衆が反発したことを認め、*c* と *z* を復活させながら *ke*・*kelki* などには *k* を保持したと述べる。*qu*、喉音の *ch*、*g* の二重発音については意見が分かれ、オクツィデンタルはフランス語同様これを認めるがノヴィアルは拒否する。ヴァールはこの問題の多くを美意識の問題と見なし、「公衆は耳よりも目に訴えられる」と言う。

**III. 派生。** ヴァールは、エスペラントもイドも国際的語彙（*-ion, -iv, -ura, -ntie* など）を規則的に派生させることができず、数千もの例外を語彙の中に隠していると論じ、ノヴィアルが動詞を自然な派生形に反して *a* クラスに押し込める傾向を批判する。イェスペルセンはこの指摘を認めて語彙を見直すと約束するが、オクツィデンタルは動詞名詞に競合する接尾辞が多すぎる（十個を挙げる）こと、形容詞から動詞を作る方法が七通りあることを学習者の混乱につながると反論し、ノヴィアルの規則的接尾辞 *-o* と *-eso* を擁護する。ヴァールは、この多様性は自然言語を映したものであり、公衆の実際の学習は厳密な論理ではなく類推によるのだと答える。

**IV. 語形変化。** 両言語は複数形 *-s*、比較級 *plu/minu*、任意の形容詞語尾 *-i*、助動詞による動詞活用を共有する。ヴァールはノヴィアルの *ha ama*（助動詞＋語幹）と動詞主題の過負荷を批判する。イェスペルセンはすでにその方式を捨て *ha convictet* のような形に移行することを決めており、任意の不定詞語尾 *-r* も受け入れると表明する。

**V. 代名詞・副詞・否定。** ヴァールは格形式の区別（*tu/te*）と所有形容詞 *nor, vor* を予期された自然な発展として擁護しつつ、ノヴィアルの *nusen, vusen* や *-n* 属格を人工的として拒否する。イェスペルセンは任意の対格 *-m* が実際には不要と判明したことを認め、オクツィデンタルの単一形 *su* に対してノヴィアルの規則的三性体系（*-e, -o, -a* と中性 *-um*）を擁護し、否定辞として *ne* ではなく *non* を採用する根拠として自身の否定研究を挙げる。ヴァールは、どの言語でも最頻出語は不規則だが流暢だと答え、*ín-* を持つ国際語はそう簡単に廃止できないと述べる。

**結論。** 両著者は、ノヴィアルが体系的規則性を、オクツィデンタルが弾力的利便性を重視することで一致し、未来の国際語は両原理の実践的統合——「ノヴィアルとオクツィデンタルからそう遠くない」もの——になると認める。

## 二つの立場

**オットー・イェスペルセン（ノヴィアル）** は*学習の容易さ*を何よりも優先する。国語が難しいのはまさに不規則性と慣用句のためだから、国際語は可能な限り高い規則性と単純さを備えなければならない。一本化された名詞接尾辞（*-o*）、規則的な抽象接尾辞（*-eso*）、*-n* による機械的な所有形成、体系的な三性体系（*-e, -o, -a*、中性 *-um*）、そして最小限の派生手段——「あらゆる余剰は有害だ」。音声学者として当初は厳格な表音綴りを支持し、常にフランス語・英語・スペイン語を知らず既知語に頼れない学習者を基準に置く。彼は言語を改訂可能なものとして扱い、経験が障害と示せば変更する姿勢を貫く。討議の中でも、表音的な *k/s* 綴りと *ha ama* 構文はすでに放棄したと報告している。

**エドガー・ヴァール（オクツィデンタル）** は*自然性*を何よりも優先する。補助語は自然言語のように振る舞わなければならない——欧州言語の共通法則に従った有機的・自律的な全体として、予備知識なしに読め、造語で公衆を驚かせないものとして。イェスペルセンが一つの規則を尊ぶところでヴァールが尊ぶのは国際的語彙の忠実な再現であり、競合する多くの接尾辞（*-ion, -ntie, -ura, -ment* など）は欠陥ではなく利便性だとする。話者はそれぞれの母語との類推で選択でき、「心理法則が規則性を要求するのは未知のものに対してだけ」——どの言語でも最頻出語は不規則だが流暢なのだ。彼は、どれほど規則的であっても、いかなる自然言語にも見られない形（*nusen*, *lon/lan*, *-eso*, *ha* ＋語幹）を一貫して拒否する。

要するに、両者とも規則性*と*自然性を要件として認めている。違うのは、両者が衝突したときにどちらを優先するかである。イェスペルセンは体系を能動的に習得しなければならない学習者の側に立って解決し、ヴァールは一見して理解しなければならない読者の側に立って解決する。
